## Installation

1. **Clone the repository**

```bash
git clone https://github.com/samarthchavda/online-document-repo.git
cd online-document-repo
```

2. **Install Backend Dependencies**

```bash
cd backend
npm install
```

3. **Install Frontend Dependencies**

```bash
cd ../frontend
npm install
```

4. **Run the Application**

- Start Backend

```bash
cd ../backend
node server.js
```

- Start Frontend

```bash
cd ../frontend
npm start
```
